# Python for Data Science — 42 SP

A personal learning repository for the École 42 specialization / master's program in Data Science and Python.

## Purpose

This repository tracks my progress through exercises, mini-projects, and experiments as I develop skills in Python, data science, and software engineering best practices.

## Learning Goals

- Build a solid foundation in Python programming
- Apply data science concepts through hands-on exercises
- Develop clean, readable, and maintainable code habits
- Document learning observations alongside implementations

## Repository Structure

```
/exercises      — Short focused exercises organized by topic (ex00–ex09, ...)
/projects       — Larger, more involved projects
/notes          — Markdown notes about concepts and insights
/utils          — Reusable helper functions shared across exercises
/tests          — Standalone testing experiments and scripts
/docs           — Documentation about architectures and design decisions
```

## How Exercises Are Organized

Each exercise lives in its own folder (`ex00`, `ex01`, …) and contains:

| File | Purpose |
|------|---------|
| `main.py` | Starter implementation file |
| `README.md` | Exercise description and objectives |
| `notes.md` | Personal learning observations |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/RogerioLS/Python_for_Data_Science-42sp.git
cd Python_for_Data_Science-42sp

# (Optional) create a virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Run an exercise
python exercises/ex00/main.py
```
