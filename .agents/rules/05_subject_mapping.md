# 05 — Subject Mapping

## Python 1 — Array

Focus: arrays, image loading, slicing, zoom, rotate, color manipulation.

Common traps: shape mismatch, RGB/grayscale confusion, uncaught file errors, display requirements.

## Python 2 — DataTable

Focus: pandas loading, plotting Gapminder data, country/campus data, population parsing.

Common traps: missing labels/title, bad CSV handling, wrong country, treating strings like numbers.

## Python 3 — OOP

Focus: abstract class, inheritance, MRO/diamond trap, vector calculator, dot product.

Common traps: wrong `__dict__`, wrong class names, abstract class accidentally instantiable, missing docstrings.

## Python 4 — Data Oriented Design

Focus: manual statistics, closures, decorators, dataclasses, package/build.

Common traps: using forbidden helpers, wrong printed labels, closure state bugs, extra output.
