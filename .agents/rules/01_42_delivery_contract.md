# 01 — 42 Delivery Contract

42 grades the repository, not the conversation.

## Contract

For each exercise, verify:

- turn-in directory matches the subject;
- required files exist;
- prototypes match exactly;
- allowed functions/libraries are respected;
- expected output structure is preserved;
- invalid input does not crash unexpectedly;
- `py_compile` succeeds;
- flake8/norminette is clean or unavailable and reported.

## Deepthought mindset

Assume evaluation stops on the first serious error. One lazy uncaught exception can sink the exercise. Cute.
