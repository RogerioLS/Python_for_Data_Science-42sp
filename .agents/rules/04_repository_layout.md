# 04 — Repository Layout

Keep the repository obvious.

## Preferred module grouping

- `python_1_array/`
- `python_2_datatable/`
- `python_3_oop/`
- `python_4_dod/`

The actual subject turn-in directories still win: `ex00/`, `ex01/`, etc.

## Do not commit

- `__pycache__/`
- `.pytest_cache/`
- virtualenvs;
- temporary images;
- generated junk;
- local absolute path config.
